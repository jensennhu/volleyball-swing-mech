üöÄ QUICK START COMMANDS
------------------------

PYTHON:
-------
1. Install: pip install -r requirements.txt
2. Run: python example_usage.py
   (Edit VIDEO_PATH in file first)
3. Or use directly:
   from volleyball_spike_analyzer import analyze_spike_biomechanics
   fig, data = analyze_spike_biomechanics('video.mp4')

R:
--
1. Install: install.packages(c("tidyverse", "ggplot2", "gridExtra", "signal", "pracma"))
2. Run: Rscript example_usage.R
   (Edit DATA_PATH in file first)
3. Or use interactively:
   source("volleyball_spike_analyzer.R")
   data <- load_pose_data("pose_landmarks.csv")
   results <- analyze_spike_biomechanics(data)
